# samples

Voice demos and finished audiobooks produced by vorpal.

## Structure

```
clips/      short voice audition clips (~2–3 MB WAV each)
finished/   complete audiobooks (.m4b) — gitignored, referenced below
```

## Voice audition clips — Trotsky *Military Writings*

Three voices auditioned against four passage types from the same source material.

| Voice | Description |
|-------|-------------|
| `blend_deep_steady` | Fenrir 55% + Michael 45% — chosen for production |
| `am_fenrir` | Fenrir single — deep, commanding |
| `bm_george` | George single — distinguished British male |

Passage types: `address_to_soldiers`, `analytical`, `peroration`, `polemical_opening`
